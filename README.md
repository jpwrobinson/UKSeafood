# Navigating sustainability and health trade-offs in global seafood systems

Data and code for Robinson JPW, Garrett A, Esclapez JCP, Maire E, Parker RWR and Graham NAJ. Navigating sustainability and health trade-offs in global seafood systems (2022) *Environ. Res. Lett.* 

To access data and re-create figures, clone or download this repository and run ```source('source_all.R')``` in R.

Ensure packages are installed by running 

```
install.packages('pacman')
pacman::p_load(ggrepel, ggradar, tidyverse, scales, janitor, cowplot, readxl, install=TRUE)
```
